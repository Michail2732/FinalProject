# Консольный многопользовательский чат
## Основные функции
 - Получение списка пользователей;
 - Регистрация пользователя;
 - Вход в чат;
 - Выход из чата;
 - Отправка сообщений;
 - Получение списка сообщений;
 - Получение текущего пользователя.
## Дополнительные функции
 - Удаление пользователя из чата.
